3. Module 1 — Time & Schedule Management
3.1 Unified Calendar
The Calendar module provides a single, merged view across Google Calendar, Outlook/Exchange, and Apple Calendar. All reads happen in real-time; writes are sent back to the source calendar automatically.
Views
	•	Day view with hourly time blocks and a vertical task sidebar
	•	Week view with drag-and-drop scheduling
	•	Month view with colored event density indicators
	•	Agenda view (upcoming 14 days list)
3.2 Time Blocking
Users can drag any task from the Task Panel directly onto the calendar to schedule it. This creates a 'time block' event synced back to Google/Outlook Calendar as a real event (preventing others from booking you during focus time).
	•	Minimum block size: 15 minutes
	•	Default block size: auto-calculated from task estimated duration
	•	Snapping: blocks snap to 15-minute intervals
	•	Color coding: blocks inherit task project color
3.3 AI Rescheduling Engine
Behaviour
If a meeting overruns its scheduled end time by > 10 minutes (detected via Calendar event status), the AI Scheduler automatically pushes all subsequent time blocks forward, reorders by priority if needed, and notifies the user via Dynamic Island and a push notification with a one-tap 'Accept Reschedule' action.

Rescheduling logic priority order:
	•	Rule 1: Hard deadlines are never moved past their deadline
	•	Rule 2: Non-urgent tasks are moved first
	•	Rule 3: Deep Work blocks are preserved unless forced by hard constraint
	•	Rule 4: If rescheduling is impossible for the day, overflow tasks are moved to next available slot

4. Module 2 — Task & Project Management
4.1 Natural Language Input
The task input bar accepts natural language and parses it into structured task data using on-device NLP (Apple NaturalLanguage framework) combined with Claude API for complex patterns.

Parsed Output

4.2 Task Hierarchy & Dependencies
	•	Projects contain Tasks; Tasks contain Sub-tasks (max 3 levels deep)
	•	Dependencies: Task B marked as 'blocked by Task A' — B is grayed out until A is marked complete
	•	Critical Path highlighting: the shortest path to project completion is highlighted in Gantt view
	•	Overdue dependency alert: if Task A is overdue and Task B depends on it, notify user immediately
4.3 Views
Kanban Board
Columns: Backlog → In Progress → Blocked → Done. Cards show priority color, due date, and estimated time. Drag cards to change status. Collapse columns on iPhone for single-column scroll mode.
Gantt Chart
Horizontal timeline showing all project tasks, their duration, and dependency arrows. Pinch-to-zoom for day/week/month granularity. Only available in Pro tier due to rendering complexity.
4.4 Priority Matrix (Eisenhower)
Tasks are automatically scored on two axes — Urgency (based on due date proximity) and Importance (based on user-assigned importance tag or AI inference from keywords like 'client', 'exam', 'deadline'). The matrix is displayed as a 2x2 quadrant view and drives the default sort order in list view.


5. Module 3 — Finance & Wealth Tracker
5.1 Account Aggregation
Bank accounts and credit cards are linked via Plaid (US/Canada) and Salt Edge (India, EU, UK). All financial data is read-only. No payment initiation. Tokens stored in iOS Keychain, never on our servers.
Supported Account Types
	•	Checking & savings accounts
	•	Credit cards
	•	Investment accounts (stocks, mutual funds)
	•	Crypto wallets via read-only API keys (Coinbase, Binance, WazirX for India)
	•	UPI-linked accounts via Salt Edge (India)
5.2 Expense Categorization
Transactions are auto-categorized using a combination of merchant name matching (rule-based), MCC codes, and a CoreML on-device classifier trained on common Indian and international merchants.

5.3 Budget Guardrails
Monthly or weekly budget limits per category. The system triggers notifications at 50%, 80%, and 100% of the budget. Budget alerts appear as Live Activities on the Dynamic Island for high-priority limits.
	•	Rollover option: unspent budget rolls into next month
	•	Shared budgets: for couples or teams (via iCloud Family Sharing invitation)
	•	Emergency pause: one tap to pause all non-essential budget tracking temporarily
5.4 Investment Snapshot
A single read-only dashboard showing total net worth, asset allocation (pie chart), and daily P&L. Broken down by: Cash & Savings, Stocks & MFs, Crypto, Other assets (manual entry).
5.5 Subscription Overload Detection
AI Insight
If a recurring subscription transaction is detected and there has been no task, note, or calendar event referencing the service in the past 30 days, the app flags it as 'Potentially Unused' and prompts: 'You haven't used Netflix-related tasks or notes this month. Still want to keep this subscription?'


6. Module 4 — Knowledge & Notes
6.1 Quick Capture
The fastest path to capture a thought. Target: captured in under 3 seconds from any context.
	•	Dynamic Island long-press — opens a floating capture bar without leaving current app
	•	Siri Shortcut — 'Hey Siri, quick note in LifeOS: [text]'
	•	Voice memo — 1-tap mic button transcribes via Apple Speech framework, saves as note
	•	Home Screen widget — single text field, tap and type, auto-saves
	•	Action Extension — share any webpage, screenshot, or text from any app into LifeOS
6.2 Bi-directional Linking
Notes, tasks, and calendar events can reference each other using [[double bracket]] syntax or the @ mention picker. Links are bi-directional — a note linked to a task shows up in that task's detail view as 'Related Notes', and vice versa.
	•	Example: Note 'System Architecture Plan' linked to Task 'Build REST API' — both show the connection
	•	Graph view (Pro only): a visual network map of all linked items
6.3 Web Clipper
Via iOS Share Sheet Extension. Clips title, URL, selected text, and optionally a full-page screenshot. Saves into the Notes module with the source URL preserved. Supports clipping to a specific project or note for context.


7. Advanced AI Scenarios
7.1 Burnout Detection
The AI monitors task completion patterns. If a user has missed 3 or more deadlines within a single day, the system detects the overload pattern and presents a proactive intervention dialog.
UI Behaviour
A card appears: 'You seem overloaded today — you've missed 3 deadlines. Should I move all non-urgent tasks to tomorrow and protect the rest of your afternoon?' with options: [Yes, Reschedule] [Let me choose] [Not now]. This action is reversible with one tap within 60 seconds.
7.2 The Group Trip Mode
When a user creates a Project tagged 'Travel', the app activates a multi-domain coordination mode:
	•	Time module: imports flight/hotel confirmation emails (Mail app integration) into the calendar automatically
	•	Tasks module: generates a smart packing list based on destination, trip duration, and user's past travel projects
	•	Finance module: creates a shared expense pool where costs can be assigned to trip participants and split evenly or by custom ratios
	•	Finance module: real-time FX rates displayed for international destinations
7.3 Context Switching via GPS
Using CoreLocation geofences, the app detects when the user arrives at or departs from saved locations (University, Office, Gym, Home) and automatically:
	•	Surfaces the task list associated with that location profile
	•	Hides task lists from unrelated contexts (e.g., Gaming/Personal tasks hidden at University)
	•	Activates the matching iPhone Focus Mode if permitted
	•	Triggers a Live Activity on the Dynamic Island showing the active context and top 3 tasks
Privacy Note
All location data is processed on-device via CoreLocation. No GPS coordinates are ever sent to LifeOS servers or stored in the cloud.
7.4 Subscription Overload Scan
Runs weekly as a background intelligence job. Cross-references recurring Finance transactions with activity across all modules. Flags orphan subscriptions — ones with no linked tasks, notes, or calendar events. Presents a monthly 'Subscription Audit' card with recommended cancellations and estimated annual savings.

10. Privacy & Security
	•	Sign in with Apple as primary authentication — no passwords stored by LifeOS
	•	All financial API tokens stored in iOS Keychain — never transmitted to LifeOS servers
	•	Secure data sync via Firebase Firestore — protected by strict Firestore Security Rules
	•	On-device ML for NLP, expense categorization, and GPS geofencing — no raw personal data leaves the device
	•	Plaid/Salt Edge tokens expire automatically; users can revoke anytime from Settings
	•	App Tracking Transparency (ATT) compliant — no cross-app tracking
	•	DPDP Act 2023 (India) and GDPR compliant data handling
	•	Annual third-party security audit
